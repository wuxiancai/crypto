### C. 60% 胜率目标需要新增过滤

要冲击 60% 胜率，建议新增规则后再回测：

| 过滤方向 | 规则建议 | 原因 |
| --- | --- | --- |
| 只做 TREND_PULLBACK | 禁用或提高 `REVERSAL_PROBE` 门槛 | 之前最优 2 年组合里 REVERSAL_PROBE 胜率低且净亏 |
| 强趋势过滤 | `ADX >= 22/25/28` 且 DI 差值超过阈值 | 过滤震荡伪趋势 |
| RR 提前过滤 | 入场前要求计划 RR >= 2.2/2.5/3.0 | 用交易次数换胜率和利润因子 |
| 回踩质量过滤 | 15m 影线触达 EMA 后，收盘必须重新回到趋势方向侧 | 减少只扫到 EMA 区域的假触发 |
| 分侧验证 | LONG-only、SHORT-only、SHORT 更严格 | 多空表现可能不对称 |
| 波动过滤 | ATR pct 过低或过高都不交易 | 避免低波动磨损和极端波动止损 |
| 连续亏损暂停 | 连续 3 笔亏损后暂停 N 根 15m K 线 | 降低震荡期连续止损 |

建议第一批新增规则回测：

1. `EMA18 / MA40, ATR12, DMI12, Swing20, TREND_PULLBACK only, ADX>=25, RR>=2.2`
2. `EMA15 / MA60, ATR14, DMI12, Swing20, TREND_PULLBACK only, ADX>=25, RR>=2.2`
3. `EMA12 / MA50, ATR12, DMI12, Swing30, TREND_PULLBACK only, ADX>=25, RR>=2.5`
4. `EMA15 / MA60, ATR12, DMI12, Swing36, TREND_PULLBACK only, ADX>=28, RR>=2.5`
5. `EMA18 / MA40, ATR12, DMI12, Swing30, LONG-only, ADX>=25, RR>=2.2`

这些是待验证组合，不能提前宣称能达到 60%。它们的目标是把当前 43%-47% 胜率压缩成更少、更高质量的交易集合。

## 推荐执行顺序

1. 先按 1 年口径补跑密集搜索，不混入 2 年结果排名。
2. 把 1 年 Top 10 统一跑 2 年，检查年化收益和回撤。
3. 对 Top 组合做策略拆分：`TREND_PULLBACK only`、`REVERSAL_PROBE only`、`LONG only`、`SHORT only`。
4. 再新增高胜率过滤：ADX 阈值、DI gap、RR 阈值、ATR pct、连续亏损暂停。
5. 最后用 3 个月、6 个月、1 年、2 年滚动窗口做稳定性验证，每个周期单独排名。

## 最终判断

修正口径后，当前 1 年收益最高组合是 Run 272：`EMA18 / MA40, ATR12, DMI12, Swing20, fee/risk=0, TRAILING`。

Run 166 的 2 年总收益更高，但不能和 1 年组合直接比较。它只能说明 `EMA10 / MA30` 这个区域值得继续做 2 年扩展验证。

当前没有任何已回测组合达到 60% 胜率。要达成这个目标，需要新增过滤条件并重新回测，不能靠现有参数结果直接得出。
